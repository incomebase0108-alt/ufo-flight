# UFO FLIGHT モデル作成 共通仕様（1号機 ⇄ 4号機）

3Dゲーム「UFO FLIGHT」(Three.js r128) の簡易プリミティブ製モデルを、Blender 5.1 で
作った高精細 GLB に差し替える。**single HTML に base64 で埋め込む**ため、各モデルは
独立した `.glb` として `models/` に書き出す。最終的に 1号機が HTML へ埋め込む。

## 分担
| 担当 | モデル | 出力ファイル |
|---|---|---|
| 1号機 | UFO（自機）, 戦闘機（敵） | `models/ufo.glb`, `models/fighter.glb` |
| **4号機** | **戦艦, 基地, 戦車, UFO母艦(ボス), ガンダム風メカ** | **`battleship.glb`, `base.glb`, `tank.glb`, `mothership.glb`, `gundam.glb`** |

## 作り方の鉄則（`common.py` を必ず import）
- `from common import *` で `wipe/mat/cube/cyl/cone/sphere/torus/bevel/join_all_except/export` が使える。
- 各 build スクリプトは冒頭で `wipe()`、最後に `export("xxx.glb")`。
- Blender 5.1 を headless 実行:
  `& "C:\Program Files\Blender Foundation\Blender 5.1\blender.exe" --background --python build_battleship.py`
  （`common.py` と同じ `blender/` フォルダで実行すること）

## 軸（最重要・間違えると向きがズレる）
Blender Z-up → glTF(+Y Up) は `Blender(x,y,z) → glTF(x, z, -y)`。
- **上は必ず Blender +Z**（全モデル共通）。
- **地上物（戦艦・基地・戦車）は底面/キール/接地を Blender Z=0 に揃える**。
  水平の向き（北南東西）は任意でよい（ゲームが接地面に合わせて回す）。
- 機首向きが要るのは戦闘機だけ（4号機担当外）。

## スケール（Blender 単位 = ゲーム単位 1:1）
| モデル | 寸法の目安 |
|---|---|
| battleship | 全長 ~0.60（X方向）・幅 ~0.12・キール上の高さ ~0.28・**キール Z=0** |
| base | 滑走路全長 ~2.4（X方向）・司令塔高さ ~1.0・**接地 Z=0**（大型） |
| tank | 全長 ~0.18・幅 ~0.10・高さ ~0.09・**接地 Z=0**（ゲーム側で更に0.7倍されるので素の寸法でOK） |
| mothership | **巨大**：直径 ~3.0（半径~1.5）・厚み ~1.0。中心が原点。up=+Z（円盤は XY 平面） |
| gundam | 全高 ~0.7・**足裏を Z=0**・直立。**正面（顔/胸/銃）= Blender -Y**（→glTF +Z。lookAt 追従用） |

## マテリアル命名規約（国別色の着色箇所）
ゲームは国ごとに色を変える。下記の名前を含むマテリアルだけコード側で色を差し替える。
- 名前に **`TINT_GLOW`** を含む → `material.emissive` を国別色へ（発光部。警告灯・マスト灯など）
- 名前に **`TINT_COLOR`** を含む → `material.color` を国別色へ（船体ストライプ等）
- それ以外 → GLB の見た目固定

担当モデルでの指定:
- **battleship**: マスト灯や艦尾灯など小さな発光球を `mat("TINT_GLOW_light", ...)` に。船体グレー/喫水下の赤は固定。
- **base**: 司令塔頂部の警告灯を `mat("TINT_GLOW_light", ...)` に。建物/滑走路は固定。
- **tank**: 着色なし（全マテリアル固定でよい）。

## 仕上げの推奨
- 箱物は `bevel(ob)` で角を軽く丸めると安っぽさが消える。
- 曲面（砲塔・煙突・車輪）は cyl/cone/sphere（自動 smooth）。
- ポリは控えめでOK（GLB が肥大すると base64 が膨らむ。各 200KB 以内目安）。
- 完了したら `models/*.glb` を commit & push。1号機が pull して埋め込む。

# 参考画像のテキスト記述（4号機向け・画像は hoshi さんから1号機へ届いたもの）

### ① 戦艦（battleship）= 旧日本海軍「大和」型のイメージ
- 細長い船体。**喫水線より下は赤（船底色）、上はライトグレー**の2トーン。
- **艦首は鋭く前方へ尖る**（ナイフのような前傾）。艦尾はやや角ばる。
- 中央やや後ろに**塔状の艦橋（パゴダマスト）**：箱を上に行くほど小さく数段積み、頂部に
  細い測距儀/マストが立つ。Z=0.28 くらいまでの高さ。
- 艦橋の後ろに**大きな煙突1本**（やや後傾の太い円柱）。
- **主砲塔（三連装）を3基**：艦首側に2基（前が低く後ろが高い段差配置=背負い式）、艦尾側に1基。
  各砲塔は低い円柱/箱のハウジングから**3本の砲身（細い円柱）が前方へ**。艦尾砲塔は後方を向く。
- 甲板上に小さな副砲・対空機銃（小円柱）を数個散らすとそれらしい。
- マスト頂部・艦尾に **`TINT_GLOW_light`** の小発光球。

### ② 基地（base）= 空軍基地・滑走路（俯瞰の造形物、ハリボテ調でOK）
- **非常に長い滑走路**を主役に（X方向 ~2.4）。濃いアスファルト色の長い平板、中央に白い破線。
- 滑走路に並走する**誘導路（taxiway）**と、横に広がる**駐機場（エプロン）**＝平らな白〜灰の板。
- エプロン脇に**格納庫（かまぼこ型 or 箱型）を数棟**、**司令塔ビル（高さ ~1.0 の縦長の箱）**1棟、
  頂部に細いアンテナ塔。司令塔頂部に **`TINT_GLOW_light`** の警告灯。
- 敷地のフチに**緑（植栽/芝）の点**をいくつか置くと俯瞰で基地らしくなる（参考画像が緑の丸を多数配置）。
- 全体は接地（Z=0）に水平に広がる「敷地」。立体物は司令塔・格納庫・アンテナくらいでよい。

### ③ 戦車（tank）= 陸自 90式戦車のイメージ
- **濃いグリーン＋黒褐色の迷彩**調（単色濃緑でも可）。
- 低く構えた**車体（箱）**＋左右に**履帯（キャタピラ＝長い箱、上面に転輪が透ける）**。転輪は片側6個ほど。
- 車体上に**平たく傾斜した砲塔**（前面が斜めに削げた箱/くさび）。
- **長い主砲身（細い円柱）が前方へ**水平に突き出す（砲塔前面中央）。砲身基部に防盾。
- 砲塔上に車長用ハッチの小ドーム、機銃の小円柱、後部に細いアンテナ1〜2本。
- 着色 TINT は不要（固定マテリアルでよい）。

### ④ UFO母艦（mothership）= 敵ボス・リアルで巨大
- プレイヤーUFO(直径0.32)の **約10倍** の威圧感（直径~3.0）。中心が原点・up=+Z。
- 基本は**巨大な円盤（母艦）**。上部に**司令ドーム**、下部に格納庫ハッチや**トラクタービームの発光口**。
- 外周に**回転リング**や多数の**窓・灯火**（`TINT_GLOW` 不要、母艦は固定色でよいが、所々を青白く発光 `emission` 推奨）。
- メカニカルなパネル分割・砲塔の張り出しなどで「兵器然」とした重厚感を。SF映画のマザーシップ風。
- ポリは増えがちなので 250KB 程度を上限目安に（窓は発光マテリアルの面で表現、穴あけ不要）。

### ⑤ ガンダム風メカ（gundam）= 人型の敵
- **オリジナルの人型モビルスーツ風メカ**（商標回避のため特定作品の意匠そのままにはしない、"風"でOK）。
- 配色は白基調＋青／赤＋黄のダクト、頭部に**V字アンテナ**・一つ目/バイザー型の発光する目（緑 or 青）。
- 直立ポーズ。胴・腰・左右の腕（肩/上腕/前腕）・左右の脚（腿/脛/足）・頭。**右手にビームライフル**（細長い箱）。
- **正面（顔・胸・銃口）を Blender -Y** に向ける（→glTF +Z、ゲームの lookAt でプレイヤーへ正対）。**足裏を Z=0**。
- 着色 TINT は不要（固定マテリアル）。全高 ~0.7。
